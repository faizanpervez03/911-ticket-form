## 📋 What Each File Does

### agentSidebar.jsx
Shows agent info on right side. Displays name, status, session ID, and logout button.

### AgentNameInput.jsx
Initial login screen. Agent enters their name before accessing form.

### CallInfoForm.jsx
First form step. Collects caller identification and contact information.

### CaseNatureForm.jsx
Second form step. Selects incident type and captures detailed description.

### DispositionForm.jsx
Third form step. Records how incident was resolved and completed.

### FormStepper.jsx
Visual progress bar. Shows user which step they're on.

### TabNavigation.jsx
Department selector. Allows switching between Police, Rescue, etc.

### FormPage.jsx
Main container. Manages all form state, navigation, and logic.

### App.jsx
Root component. Renders FormPage.
